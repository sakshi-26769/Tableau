# Tableau
📊 Sales Comparison Dashboard 
This project presents an interactive Tableau dashboard designed to compare sales performance between Primary and Secondary regions. The dashboard provides a comprehensive view of business performance using key metrics, trends, and geographic insights.

🔍 Key Features:
📌 Comparison of Primary vs Secondary region sales
📊 KPIs including:
Total Sales
Average Sales per Order
Number of Customers
Number of Orders
Number of Products Sold
📈 Sparklines to visualize sales trends over time
📦 Sub-category analysis using bar charts
🗺️ Geographic maps to understand regional distribution
📅 Tracking of first order dates for both regions

Project 2 
🚔 Crime Analysis Dashboard (Tableau Project)

This project presents an interactive Tableau dashboard designed to analyze crime patterns across a city. It provides insights into the types, frequency, and distribution of crimes over multiple years, helping to understand trends and identify high-risk areas.

🔍 Key Features:
📊 Crime Type Analysis:
Visual comparison of different crime categories such as theft, battery, assault, burglary, and more.
📈 Year-wise Trend Analysis:
Line chart showing the percentage of crime incidents reported over several years to identify increasing or decreasing trends.
🗺️ Geographical Visualization:
Map-based representation of crime occurrences across different locations in the city.
🔵 Most Common Crimes:
Bubble chart highlighting the most frequently reported crimes, making it easy to identify dominant crime types.
📅 Multi-year Comparison:
Analysis of crime data from different years to observe patterns and changes over time.
